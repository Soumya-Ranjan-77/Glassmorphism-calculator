# Glassmorphic Calculator

Live At:https://glassmorphism-calculator-pd4q.onrender.com

A modern and visually appealing calculator built using **HTML**, **CSS**, and **JavaScript**. This project features a beautiful **Glassmorphism UI design**, creating a frosted-glass effect with smooth transparency, blur effects, and clean styling. The calculator is capable of performing basic arithmetic operations such as addition, subtraction, multiplication, and division with an intuitive and responsive user interface.

## Features

* Elegant Glassmorphism design
* Responsive layout for different screen sizes
* Basic arithmetic operations (+, −, ×, ÷)
* Clear and Delete functionality
* Real-time display updates
* Smooth button hover and click effects
* Built using pure HTML, CSS, and JavaScript
* No external libraries or frameworks used

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Goal

The primary objective of this project is to practice front-end web development concepts while creating a functional and attractive calculator application. It demonstrates DOM manipulation, event handling, user input processing, and modern UI design techniques using Glassmorphism.

This project is suitable for beginners who want to strengthen their understanding of JavaScript fundamentals and improve their web design skills through hands-on development.
